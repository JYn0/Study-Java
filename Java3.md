## 연산자(operator)

* 연산자(operator) : 연산을 수행하는 기호
* 피연산자(operand) : 연산자의 작업 대상



|    종류     |          연산자          |
| :---------: | :----------------------: |
| 산술 연산자 |  +  -  *  /  %  <<  >>   |
| 비교 연산자 |   >  <  >=  <=  ==  !=   |
| 논리 연산자 | &&  \|\|  !  &  \|  ^  ~ |
| 대입 연산자 |            =             |
|    기타     |  (type)  ?:  instanceof  |

* 두 피연산자의 타입을 같게 일치시킨다.(보다 큰 타입으로 일치)
* 피연산자의 타입이 int보다 작은 타입이면 int로 변환된다.
